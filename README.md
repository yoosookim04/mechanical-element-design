# MED_Project — 기계요소설계 (Mechanical Element Design)

50 kW급 UTV(Utility Task Vehicle) 2단 변속기를 대상으로, 축·기어·베어링·클러치를 AGMA/Shigley 기준에 따라 해석·최적설계하는 MATLAB 프로젝트입니다. 범용 응력/안전계수 함수 라이브러리(`+stress`, `+static`, `+AGMA`, `+gear`)와, 이를 사용하는 요소별 설계 스크립트로 구성되어 있습니다.

## 설계 대상 시스템

- **동력**: 50 kW, 입력 6000 rpm
- **변속**: 2단 (목표 기어비 1단 ≈2.0, 2단 ≈1.5), 습식 다판 클러치로 단 전환
- **축 구성**: 상축(입력, 중실) — 하축(출력, 중공), 두 단의 기어가 한 입력축-출력축 위에서 중심거리를 공유
- **요구 수명**: B10 5000 h

## 단위계
`[mm, N, MPa, N·mm, m/s]` (클러치 설계 일부는 SI 단위 `[m, kg, Pa]` 사용)

## 최종 설계 결과 (Final Design Summary)

## 폴더 구조

```
+stress/         범용 응력·단면 함수 (중실/중공 통합)
+static/         정적 파손이론 (DE, MSS) 기반 안전계수
+AGMA/           AGMA 기준 기어 굽힘·접촉 강도평가
+gear/           기어 간섭(언더컷) 회피 최소 피니언/기어 잇수
gear_design/     기어 설계 스크립트 (모듈 공유, 헬릭스각 최적화 등 버전별)
shaft_design/    축 설계 스크립트 (처짐, 비틀림각, 임계속도, 단차축 검증)
gear_design_final.m        기어 최종 설계 (1·2단 모듈 독립 선정 + 헬릭스각 최적화)

shaft_design_step_added.m  단차축(stepped shaft) 검증
bearing_selection.m        축 반력 기반 베어링(DGBB/CRB) 수명(L10h) 계산
clutch_design.m            습식 다판 클러치 다목적 최적설계 (토크·열·마모·드래그)
```

### `+stress/` — 응력 및 단면 계산
중실축은 인자 4개, 중공축은 외경/내경 비율 `x`를 추가로 전달 (`nargin < N` 분기로 통합).

| 함수 | 설명 |
|---|---|
| `bending_stress(M, d, x)` | 굽힘 인장응력 σ_b |
| `torsion_stress(T, d, x)` | 비틀림 전단응력 τ |
| `axial_stress(F, d, x)` | 축방향 응력 |
| `vonMises(σ_x, τ_xy)` | 등가응력 (응력 입력) |
| `vonMises_MT(M, T, d, x)` | 등가응력 (하중 입력) |
| `tau_max(σ_x, τ_xy)` | 최대전단응력 (응력 입력) |
| `tau_max_MT(M, T, d, x)` | 최대전단응력 (하중 입력) |
| `principal_stress(σ_x, τ_xy)` | 주응력 σ_1, σ_2 |
| `eq_moment(M, T)` | 등가굽힘모멘트 M_e |
| `eq_torque(M, T)` | 등가비틀림모멘트 T_e |
| `area(d, x)` | 원형 단면 면적 |
| `I_solid(d)` / `I_hollow(d, x)` | 단면 2차 모멘트 |
| `I_rectangular(b, h)` | 직사각형 단면 2차 모멘트 |

### `+static/` — 정적 파손이론
연성 재료 기준 안전계수. **응력 입력**과 **하중 입력** 두 가지 인터페이스 제공.

| 함수 | 입력 | 출력 |
|---|---|---|
| `SF_DE(σ_x, τ_xy, S_y)` | 응력상태 | DE(왜곡에너지) 이론 n |
| `SF_MSS(σ_x, τ_xy, S_y)` | 응력상태 | MSS(최대전단응력) 이론 n |
| `SF_DE_MT(M, T, d, S_y, x)` | 하중·형상 | DE 이론 n |
| `SF_MSS_MT(M, T, d, S_y, x)` | 하중·형상 | MSS 이론 n |
| `SF(σ_x, τ_xy, S_y)` | 응력상태 | `[n_DE, n_MSS]` 동시 반환 + 콘솔 출력 |
| `SF_MT(M, T, d, S_y, x)` | 하중·형상 | `[n_DE, n_MSS]` 동시 반환 + 콘솔 출력 |

### `+AGMA/` — AGMA 기어 강도평가
스퍼·헬리컬 기어의 굽힘(Bending) 및 접촉(Contact) 안전계수를 산출.

**전제 조건**:
- 재료: 침탄강 (Eh), `Z_E = 191`
- 정밀도 등급: `Q_v = 6` (`K_v` 내부 고정)
- 신뢰도: 0.99 (`Y_θ = Y_Z = S_F = S_H = 1`)
- 기준 응력값: `S_t = 448 MPa` (굽힘, 10⁷ cycles), `S_c = 1551 MPa` (접촉, 3×10⁶ cycles)

**보정계수**

| 함수 | 설명 |
|---|---|
| `K_v(v)` | 속도계수 (선속도 m/s 입력) |
| `K_s(m_n, β, b, N_p, N_g)` | 크기계수 (Lewis form factor 기반) |
| `K_H(m_n, β, b, N_p)` | 하중분포계수 |
| `Y_J(β, N_p, N_g)` | 굽힘 형상계수 (Shigley Fig. 14-7, 14-8; β=0 시 스퍼기어 등가) |
| `Z_I(m_n, α_n, N_p, N_g)` | 접촉 형상계수 (스퍼기어) |
| `Z_I_helical(m_n, α_n, N_p, N_g, β)` | 접촉 형상계수 (헬리컬기어) |
| `Y_N(N, u)` / `Z_N(N, u)` | 굽힘·접촉 stress-cycle 계수 (피니언/기어 동시 산출) |
| `form_factor(N_p, N_g)` | Lewis Form Factor Y (Shigley Table 14-2) |

**응력 및 안전계수**

| 함수 | 설명 |
|---|---|
| `sgm(W_t, v, m_n, α_n, β, b, N_p, N_g)` | 실제 굽힘응력 σ (피니언·기어) |
| `sgm_all(N, u)` | 허용 굽힘응력 σ_all |
| `sgm_c(W_t, v, m_n, α_n, β, b, N_p, N_g)` | 실제 접촉응력 σ_c (기어쌍 단일값) |
| `sgm_c_all(N, u)` | 허용 접촉응력 σ_c,all |
| `S_F(N, W_t, v, m_n, α_n, β, b, N_p, N_g)` | **굽힘 안전계수** (피니언·기어) |
| `S_H(N, W_t, v, m_n, α_n, β, b, N_p, N_g)` | **접촉 안전계수** (피니언·기어) |

|---|---|
| `interference_free_p(α_n, β, u)` | 언더컷이 발생하지 않는 최소 피니언 잇수 |

## 요소별 설계 스크립트

### 기어 — `gear_design_final.m`, `gear_design/`
1단·2단 기어쌍의 모듈·잇수·헬릭스각을 결정하는 전수탐색 + `fmincon` 최적화.
- 두 단이 입력축-출력축 위에서 **중심거리를 공유**하므로, `m_n1·Σ1/cosβ1 = m_n2·Σ2/cosβ2` 등식으로 β2를 β1에서 유도
- 잇수 탐색 단계에서 간섭(언더컷) 회피, 피니언·기어 잇수 서로소(hunting tooth 방지) 제약 적용
- 연속변수(β1, d/b 비율)는 `fmincon`(SQP)으로 블랭크 부피 최소화
- 제약: 굽힘/접촉 안전계수 하한, 접촉비 하한, 전달하중 상한, 베타 범위
- `+AGMA`, `+gear` 패키지를 직접 호출하여 강도·간섭 평가

### 축 — `shaft_design_not_stepped.m`, `shaft_design_step_added.m`, `shaft_design/`
- `shaft_design_not_stepped.m`: 상축(중실)·하축(중공) 외경, 중공비, 기어 축방향 위치(xG)를 `fmincon`으로 동시 최적화하여 질량 최소화. 정적/피로 안전계수, 처짐비, 비틀림각, 임계속도(危險速度) 제약을 만족하는 배치를 탐색하고 결과를 `.mat`으로 저장
- `shaft_design_step_added.m`: 위 결과(`shaft_fmincon3_layout_result.mat`)를 불러와, 실제 적용할 **단차축(stepped shaft)** 치수를 직접 입력해 강도·처짐·임계속도·베어링/클러치 공간 조건을 최종 검증하고 베어링 담당자 전달값(반력 envelope 등)을 출력
- `shaft_design/` 하위 함수: `shaft_load`(기어 전달하중 3성분), `bending_deflection`, `torsion_angle`, `shaft_stress`, `critical_speed`

### 베어링 — `bearing_selection.m`
상축/하축 각 2개소(DGBB 고정측 + CRB 유동측) 배치에서, 단별 반력·rpm으로부터 동등가하중 P와 L10h 수명을 계산하여 요구수명(5000 h) 대비 PASS/FAIL 판정.

### 클러치 — `clutch_design.m`
습식 다판 클러치를 MINLP 방식(정수 판수 nf는 반복문, 연속변수는 `fmincon` SQP)으로 최적화.
- 목적함수: 질량·부피·드래그손실(효율) 가중합
- 제약: 토크용량, 슬립속도, 평균 열유속, 마모수명(B10 5000h 역산 마모여유) 만족

## 사용 예시

```matlab
% 정적 파손 평가 (중실 원형축, 굽힘 + 비틀림)
[n_DE, n_MSS] = static.SF_MT(M=120e3, T=80e3, d=40, S_y=400);

% 헬리컬 기어 안전계수
[S_F_p, S_F_g] = AGMA.S_F(N=1e8, W_t=2500, v=5.2, ...
                          m_n=2.5, alpha_n=20, beta=15, ...
                          b=30, N_p=20, N_g=36);
[S_H_p, S_H_g] = AGMA.S_H(1e8, 2500, 5.2, 2.5, 20, 15, 30, 20, 36);
```

설계 스크립트는 각 파일을 직접 실행(run)하는 방식이며, `gear_design_final.m` → `shaft_design_not_stepped.m` → `shaft_design_step_added.m` → `bearing_selection.m` 순으로 앞 단계 결과(기어 치수, 축 배치)를 다음 단계 입력으로 사용합니다.

## 적용 범위 (Scope)
- **대상**: 원형 단면 회전축, 스퍼/헬리컬 기어, 베어링, 습식 다판 클러치 등 회전축계 기계요소
- **응력 상태**: 평면응력 (σ_y = 0) — 굽힘응력 + 비틀림전단응력 조합
- **재료**: 인장·압축 강도가 비슷한 **연성 재료** (덕타일 강) 기준
  - 취성 재료 파손이론(BCM, MNS, DCM)은 본 라이브러리 범위 외
- **권장 외 범위**: 하우징, 압력용기 등 이축응력(σ_y ≠ 0)이 지배적인 구조물 → FEA 적용 권장

## 향후 작업
- `+fatigue/` : 피로 파손이론 (Goodman, Soderberg, Gerber)
- `+bearing/` : 베어링 수명 평가 패키지화 (현재 `bearing_selection.m` 스크립트 내 직접 구현)
- AGMA 재료/정밀도 등급 매개변수화 (현재 침탄강·Q_v=6 하드코딩)
- 클러치 설계 내 `*` 표시 가정값(관성, 듀티, 마모율 등)을 실제 데이터로 교체
